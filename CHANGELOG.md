## [1.0.0] - 2026-03-29

### Features

- **api:** JWT authentication with email verification and password reset
- **api:** Google OAuth 2.0 login via Passport
- **api:** Role-based access control (`user`, `curator`, `admin`)
- **api:** Worker CRUD with curator-gated listings
- **api:** Profile image uploads with Multer and Sharp (method-override for multipart PUT)
- **api:** Category management with seed data
- **api:** Admin endpoints for user and worker management
- **api:** Rate limiting, CORS, and Helmet security headers
- **api:** Pino structured logging
- **api:** Prisma ORM with PostgreSQL (User, Worker, Category, Location models)
- **api:** Nodemailer email transport with HTML templates
- **contracts:** Registry Soroban contract — curator-gated worker registration, toggle, update, deregister
- **contracts:** Market Soroban contract — direct tips and time-locked escrow payments
- **contracts:** Protocol fee (max 5%) with configurable fee recipient
- **contracts:** On-chain privacy via SHA-256 hashed location and contact fields
- **contracts:** Automatic storage TTL extension (~1 year) on every write
- **contracts:** In-place WASM upgrade support for both contracts
- **app:** Next.js 14 frontend scaffold with Tailwind CSS and shadcn/ui
- **ci:** GitHub Actions workflows for API tests, contract build/clippy, and app lint/build
